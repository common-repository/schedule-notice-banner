== Description ==
Schedule Notice Banner is a WordPress plugin that allows users to schedule the display of a banner before or under the navigation area of their website. Users can set the start date and expiry date for the banner.
The image will be removed automatically at the set period.

== Installation ==
From the WP admin panel, click \"Plugins\" -> \"Add new\".
In the browser input box, type \"Schedule Notice Banner\".
Select the \"Schedule Notice Banner\" plugin (authored by \"Toyin Alagbe\"), and click \"Install\".
Activate the plugin.
OR...

Download the plugin from this page.
Save the .zip file to a location on your computer.
Open the WP admin panel, and click \"Plugins\" -> \"Add new\".
Click \"upload\".. then browse to the .zip file downloaded from this page.
Click \"Install\".. and then \"Activate plugin\".
OR...

Download the plugin from this page.
Extract the .zip file to a location on your computer.
Use either FTP or your hosts cPanel to gain access to your website file directories.
Browse to the wp-content/plugins directory.
Upload the extracted Schedule-notice-banner folder to this directory location.
Open the WP admin panel.. click the \"Plugins\" page.. and click \"Activate\" under the newly added \"Schedule Notice Banner\" plugin.
